# React + TypeScript + Vite

## Архитектура проекта

- **pages** — файлы с общей разметкой страниц
- **components** — переиспользуемые части сайта
  - **components/ui** — общие UI-компоненты (кнопки, ссылки и т.д.)
- **routers** — маршрутизация сайта
- **store** — конфигурация и настройка стейт-менеджера
  - **store/slices** — настройка и конфигурация отдельных стейтов
- **services** — настройка запросов к API
- **types** — общие типы для тел запроса, структуры стейта и т.д.
- **utils** —
